In which parcel am I?

Marius Huber - ITAS

2026-04-21

Purpose

This website is created by the ITAS-project “CitiesForALL”. The purpose of it is to show in which parcel the visiter is momentarily. This serves the need of accuracy in putting camera traps only in parcel numbers that we have a permission to use as camera locations.

Necessary packages

The packages I activated are ggmap, sf, leaflet, leaflet.extras, leaflet.extras2, stars, tidyverse, mapview, terra, and htmltools - some of them might not be needed, however.

Map

The parcels of relevance are shown in blue, if we have the permission to install a camera in the parcel, in red, if we have not!
